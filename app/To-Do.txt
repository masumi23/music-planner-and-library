Library
    - keyboard shortcut for copying songs
    - Export data option
    - Numbering how many songs are under each tag
    - Search function (automatically highlights the first song and you can navigate up and down with arrow keys. Hit enter to select that song)
    - Auto-complete textualSource
Class Planning 
    - Specific Course information
        + Knowledge items
            * Knowledge items are categorized as "rhythmic", "melodic", "theory", "skills", etc.
        + All songs
            + list of all songs already planned (and perhaps that I want to use, but not planned yet)
        + Classes - identified by number rather than date
    - 1) Flow chart: Macro
        - Description
            + x axis is the class
            + y access is a list of knowledge items "nuggets".
            + Information written at the intersection will be stored as the behavioral objective for that nugget and class
        - Later features
            + Intersections are colored if they are the date a nugget is made conscious
            + Students
            + Date generator
                + Start Date
                + End Date
                + Class intervals
                + Breaks
            + Planned by month
            + Drag and drop information at intersections
            + Add songs
            + Song BO can link to knowledge item's BO's if they overlap
    - 2) Flow chart: Checklist
        - Description
            - x axis is the class
            - y axis is a list of categories to cover.
                + Songs for Singing
                + Games/Movement
                + New Song/ Game
                + Rhythmic: Prep
                + Rhythmic: Practice
                + Melodic: Prepare
                + Melodic: Practice
                + Part-Work
                + Reading/ Sight-singing
                + Writing/ Dictation
                + Listening
                + Form, Inner Hearing, Improv, Comp, Other
            - The intersection conatains multiple fields:
                * Song (ex. Liza Jane) or activity (ex. tone ladder)
                    - There is a button to chose whether it is a song or an activity. Songs come from a drop down list.
                * nugget (Placed in category Rhymic: Prep - can be entered in the macro-planning view)
                * behavioral objective (can be entered in macro-planning view)
        - Later feature
            - Ability to enter new songs on the spot
            - Right click (or something else) on songs to see their stats:
                + How many times used in the course
                + Last time used
                + % of classes it is in
            - When filling in songs associated with teaching an element can get suggestions which will show a small list of songs that are good for teaching that element (sorted from the library, prioritizing songs thats that have already been selected for the course)
            + Sort y access according to catagories, or according to class order (y access is no longer the category but the song order)
    - 3) Single Class
        + Description
            * x axis is fields for the specific songs (same as from checklist view and song library)
                * CSP
                * tone set
                * props
                * song title 
                * [checklist category]: [nugget] 
                * behavioral objective
                * full description (optional - Review, Point, Reinforcement format)
                * shape
                * transition 
                * reflection
            * y axis is songs/ activities listed in order of presentation
    - Views for outside use
        + Print view - print the lesson plan ex. for student or teacher observation
        + Parent course view - view the course where the x axis displays class dates and the y axis shows the song order. Each song is hyperlinked to their entry in the song library. The song library view only shows the song title, attribution, general notes, and music - but not analysis.
